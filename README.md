# Lean Cheatsheet

This repository contains reference material for working with the [Lean theorem prover](https://lean-lang.org).

- **[Lean.tex](Lean.tex)** — A LaTeX cheat-sheet covering Lean syntax, tactics, and common patterns.
- **[RulesOfThumb.lean](RulesOfThumb.lean)** — A collection of example proof states you might encounter while writing Lean proofs, along with guidance on how to handle them.
